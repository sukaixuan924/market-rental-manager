# 市场日租管理系统

为市场日租位置管理者提供一款移动端+PC端通用的信息化管理工具，实现每日摊位出租情况的便捷记录与查询。

## 功能特性

- 🏪 **位置管理** - 创建、编辑、删除摊位位置
- 📅 **日历视图** - 直观的出租日历，支持月视图
- 📝 **出租记录** - 记录每日出租情况（支持日租/长租）
- 📊 **数据统计** - 收入统计与分析
- 💾 **本地存储** - 数据存储在浏览器本地
- 📱 **响应式设计** - 完美支持移动端和PC端

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **UI**: Element Plus (PC) + Vant (移动端)
- **跨平台**: Tauri 2.x
- **状态管理**: Pinia

## 快速开始

### 安装依赖

```bash
cd market-rental-manager
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建桌面端

```bash
npm run tauri build
```

### 构建网页端

```bash
npm run build
```

## 数据说明

- 数据存储在浏览器 localStorage 中
- 可在"设置"页面导出/导入数据备份
- 清除浏览器数据会导致记录丢失
